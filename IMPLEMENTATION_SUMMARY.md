## How to Add Sinhala Translations

The app is ready for Sinhala translations. Here's what you need to do:

1. **Open** `lib/l10n/app_si.arb`
2. **Translate** the English text values to Sinhala (keep keys unchanged)
3. **Run** `flutter gen-l10n` to regenerate localization code
4. **Test** by running the app and selecting Sinhala
